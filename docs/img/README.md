# Images for the README

Drop these in and the (currently commented-out) references in the top-level
README will render:

- **`demo.gif`** — a ~30-second screen capture of the demo: dragging emails
  between lanes, opening the "why?" explainability panel, clicking "ask LLM"
  to escalate a card. Keep it under ~10 MB. Then uncomment the
  `![Winnow demo](docs/img/demo.gif)` line near the top of the README.

  explainability panel showing the signed feature contributions.

The live demo at https://winnow-eight.vercel.app/demo is the primary
"see it work" surface; the GIF is a fallback for readers skimming the repo.
